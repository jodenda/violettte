Les deux types de zones graphiques dans les pages Web

Points communs :

  • La largeur et la hauteur de ces zones rectangulaires doivent être définies soit :
    dans le code CSS de la page ; exemple

    svg { width : 200px ; height : 80%; }

    dans la balise ouvrante ; exemple :


    < svg width="200" height="400" >

    dans du code javascript ; exemple :

    document.getElementById("idSVG").style.width="200"; document.getElementById("idSVG").style.height="400";
